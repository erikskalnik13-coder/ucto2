# Účtování – nativní Android aplikace (Jetpack Compose)

Tohle je nativní aplikace (ne WebView). Dataset je v `app/src/main/assets/dataset.json`.

## Build APK
1) Rozbal ZIP
2) Android Studio → File → Open (složka projektu)
3) Počkej na Gradle sync
4) Build → Build Bundle(s) / APK(s) → Build APK(s)

APK:
`app/build/outputs/apk/debug/app-debug.apk`
